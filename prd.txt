## 2. System Architecture (The "Hybrid" Model)

### 2.1 Cross-Domain Handshake
The platform operates across two primary domains with a secure JSON Web Token (JWT) handshake:
1.  **Home Site (`teachmeai-home-site`)**: Conversion-focused landing page (Vercel).
2.  **Intake App (`teachmeai-intake-app`)**: Deep-assessment conversational interface (Vercel).
3.  **Agent Service**: Backend orchestration layer (Google Cloud Run).

**The Handshake Flow**:
- User starts challenge intake on Home Site.
- Home Site signs a JWT containing `name`, `email`, and `challenge`.
- User is redirected to `intake.teachmeai.in?token=...`.
- Intake App verifies token and pre-fills session state for a seamless transition.

---

## 3. The Intelligence Hierarchy (Agent Service)
The `agent-service` is a Genkit-powered Express application that manages a 6-agent choreography:

1.  **The Guide**: Manages initial conversational extraction and lead capture.
2.  **The Profiler**: Performs expert psychometric analysis (Decision style, Uncertainty handling, Cognitive load tolerance).
3.  **Learner Dimensions**: Assesses 5 key areas: SRL (Self-Regulated Learning), Motivation, Preferences (VARK), Readiness, and Constraints.
4.  **Deep Research**: Maps industry-specific AI opportunities and identifies "Quick Win" portfolio artifacts.
5.  **The Strategist**: Calibrates raw goals into outcome-focused IMPACT goals.
6.  **The Tactician**: Assembles a weekly execution plan (Act/Check/Transform) calibrated to time and skill barriers.

---

## 4. Hyper-Personalization Logic (v2.9.0)
The system adapts the output dynamically based on four core "Pivots":

- **Seniority Pivot**: 
  - *Junior*: Focus on task-level automation and individual efficiency.
  - *Senior/Lead*: Focus on strategic leverage, team scaling, and decision support.
- **Technical Pivot (Digital Mastery)**: 
  - *Score < 3*: Suggests simple, web-based tools (ChatGPT, Claude, Perplexity).
  - *Score >= 4*: Suggests automation, APIs, custom Python scripts, and LangChain.
- **Time Calibration**: 
  - Roadmaps are time-fenced. If availability < 120m/week, sessions are 20-min "Sprints."
- **Learning Style Pivot (Kolb)**: 
  - *Activists/Pragmatists*: Start with "Building" tasks.
  - *Theorists/Reflectors*: Start with "Researching" tasks.

## 5. Functional & Feature Specs

### 5.1 Interactive Components
- **`InterviewChat`**: A real-time conversational UI with turn-by-turn state persistence.
- **IMPACT Analysis Display**: A rich UI for viewing the generated 90-day roadmap.
- **Lead Capture**: Seamless integration with **Mailerlite** for newsletter double opt-in.

### 5.2 Persistence & Operations
- **Google Sheets Sync**: All intake data + AI analysis blobs are appended to a centralized Google Sheet for administrative review.
- **Fail-Safe Logging**: Data is logged *before* high-latency AI calls to ensure no leads are lost during errors.
